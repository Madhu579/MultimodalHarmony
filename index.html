About Us

Here is some information about our platform and how to use it:

  • Text to Speech
    1. specify the type of input (file or enter text)
    2. file type - Only ".txt" file allowed
    3. Specify the language of input text
    4. Enter the name of outfile audio file you wish to download
    5. click Download audio and wait
    6. Reset - clear all inputs
  • Speech to Text
    1. Only English language is allowed
    2. Upload mp3 file
    3. audio time upto 4:30mins
    4. click convert and wait
  • Audio to Text
    1. Only English language is allowed
    2. if browser ask permission to use microphones click allow to proceed
    3. click start Recording and speak in english
    4. click stop Recording once completed
    5. to download click Download button
